# WXT + Vue 3

This template should help get you started developing with Vue 3 in WXT.

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar).


## Commands

```
npm run dev                             Start the WXT dev server (hot reload)
npm run build                           Build the extension for production
npm run curate                          Incremental — only scrapes new cards, removes deleted ones
npm run curate -- --force               Re-scrape all cards from scratch
npm run curate -- --count               Show current card total
```
